# 愤怒猫咪功能使用说明

## 功能描述
当用户在 1 秒内连续点击猫咪 5 次时，会触发特殊反应：
- 猫咪会被替换为"干嘛......"表情
- 自动播放语音："干嘛————"（使用讯飞 TTS，不经过大模型）
- 2 秒后自动恢复正常

## 当前实现
目前使用的是一个带有 😾 表情符号和文字的临时显示效果。

## 如何替换为实际图片

### 方法 1：使用图片文件（推荐）
1. 将您的图片（例如用户上传的"干嘛......"白猫表情包）保存到项目的 `public` 文件夹中
2. 命名为 `angry-cat.png` 或其他名称
3. 在 `components/CatAvatar.tsx` 中找到 `if (showAngryCat)` 部分
4. 将代码修改为：
   ```tsx
   if (showAngryCat) {
       return (
           <div className="relative w-80 h-72 mx-auto flex items-center justify-center animate-shake">
               <img 
                   src="/angry-cat.png" 
                   alt="愤怒的猫咪" 
                   className="w-full h-full object-contain rounded-2xl"
                   style={{
                       filter: 'drop-shadow(0 0 20px rgba(255, 0, 0, 0.5))'
                   }}
               />
           </div>
       );
   }
   ```

### 方法 2：使用 Base64 编码
1. 将图片转换为 Base64 编码（可以使用在线工具）
2. 在 `components/CatAvatar.tsx` 中找到 `if (showAngryCat)` 部分
3. 将 `src` 属性改为：
   ```tsx
   src="data:image/png;base64,YOUR_BASE64_STRING_HERE"
   ```

## 测试
1. 运行应用：`npm run dev`
2. 快速连续点击猫咪 5 次（1 秒内）
3. 应该看到表情变化并听到"干嘛————"的语音
4. 2 秒后自动恢复正常

## 技术细节
- 点击计数器在 1 秒后自动重置
- 使用 `directTextToSpeech` 函数直接调用讯飞 TTS API
- 不经过大模型，快速响应
- 动画使用 CSS `@keyframes shake` 实现抖动效果
